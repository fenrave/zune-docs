# Init Command
---
import { FileTree } from 'nextra/components'

The `init` command is used create files for a new project in the current directory.

### Usage

```bash
zune init [template]
```

### Templates
- \<none\> - Create a new project template.
- `bare` - Create a blank template.
- `module` - Create a module template.
