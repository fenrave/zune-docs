# Luau Command
---

The `luau` command is used to display information about the runtime's current [Luau](https://luau.org/).

### Usage

```bash
zune luau <list-fflags | version>
```

#### List Feature Flags

Displays the current visible feature flags in the runtime with their default values.

```bash
zune luau list-fflags
```

#### Version

Displays the current [Luau](https://luau.org/) version in the runtime.

```bash
zune luau version
```
